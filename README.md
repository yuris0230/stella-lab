# StellaLab（ステララボ）
## サイト概要
### サイトテーマ
架空オンラインゲームのキャラやスキルなどの攻略や話題をトピック単位で整理して議論・共有できるのコミュニティサイト
### テーマを選んだ理由
既存の攻略サイトでは、コメントが時系列に流れて話題が混在しやすく、PC閲覧時の読みづらさも課題と感じていました。
特に ビルド／スキル回し／編成相性 のような細分化された議論では、トピック分離とスレッド管理が不可欠です。
そこで、管理側の正規化されたデータベースと、ユーザー側のトピック型ディスカッションを組み合わせ、検索性・可読性・議論の質を高める場を提供するため、本テーマを選定しました。
​
### ターゲットユーザ
- アップデート直後、情報が各所に散在して追い切れず、PCでも一覧性高く性能や評価を確認したい人
- 高難度コンテンツに挑戦前で、ビルド例や立ち回りの根拠をトピック別に学びたい初心者〜上級者
- 過去ログをトピック別に学びたい初心者
- 検証勢で、ダメージ計算やスキル回しの結果を再現可能な形で共有し、フィードバックを得たい人
​
### 主な利用シーン
- パッチノート公開直後に、verX.Yタグで新キャラ/新スキル投稿を絞り込み、初見評価トピックを立てるとき
- 編成検討時に、候補装備やビルド投稿をブックマーク（いいね）で保存・比較し、グループに共有するとき
- 攻略質問があるときに、該当コンテンツのトピックでQ&Aし、解決済みフラグを付けて後から参照しやすくするとき
- イベントやバナーなどの開始・終了日をサイト内カレンダーで一覧し、周回や育成の計画を立てたいとき
​
